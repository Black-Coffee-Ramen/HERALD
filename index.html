The Detection Pipeline

A five-stage autonomous flow that transforms raw infrastructure logs into actionable security intelligence.

Ingestion

Real-time monitoring of Certificate Transparency logs and NRD feeds.

Classification

Ensemble ML analysis on 30+ lexical and behavioural features.

Visual Check

OCR fallback via headless Chromium to identify pixel-perfect clones.

Enrichment

Automated WHOIS, ASN, and IP forensics for every detection.

Notification

Instant alerts via API, Webhooks, or native Telegram integration.

Decoupled & Scalable Architecture

FastAPI Core

High-performance REST layer for management, reporting, and search.

Redis Task Queue

Asynchronous job processing to ensure high throughput under heavy load.

ML Workers

Dedicated workers running scikit-learn ensembles and headless browsers.
